1
Essential Addons for Elementor — تعارض Namespace
السبب: الإضافة تستخدم namespace Essential_Addons_Elementor\Classes وتستدعي دالة is_plugin_active_for_network() من WordPress core بدون إشارة إلى الـ global namespace. PHP بحث عنها داخل namespace الإضافة ولم يجدها.

التأثير: الموقع بالكامل لا يعمل — رسالة "خطأ فادح" (HTTP 500).

الإصلاح: تم إضافة شرطة مائلة عكسية \ قبل الدالة في السطر 46.
📄 wp-content/plugins/essential-addons-for-elementor-lite/.../Compatibility_Support.php
// قبل التعديل (يسبب خطأ):
is_plugin_active_for_network($plugin_path);

// بعد التعديل (صحيح):
\is_plugin_active_for_network($plugin_path);

🚨 المشكلة الثالثة — اختراق (Malware) — الأكثر خطورة

3
Speed Optimizer — إضافة مزيفة تحتوي على Crypto Malware
السبب: إضافة وهمية تحمل اسم "Speed Optimizer" تم تثبيتها في 9 مايو 2026. الإضافة تحتوي على كود خبيث متطور يستخدم تقنية Polygon Blockchain.

آلية عمل المالوير بالتفصيل:
1. المستخدم يفتح الموقع ← 2. حقن overlay (z-index:9999) يغطي الشاشة ← 3. الاتصال بـ Polygon blockchain
← 4. استدعاء عقد ذكي (0x31Be3fBbbC539191f4b597DDDdAd1177363B741d) للحصول على رابط سكربت
← 5. العقد يرجع رابط URL مُشفر ← 6. فك التشفير وتحميل سكربت ضار خارجي ← 7. إخفاء الـ overlay بعد 5 ثوانٍ
خباثات إضافية:
  • يكتشف نوع المتصفح تلقائيًا — يظهر "Chrome Update" أو "Edge Update" حسب المستخدم
  • يشتغل فقط على نظام Windows لاستهداف أكبر عدد من الضحايا
  • يتجاهل تمامًا البوتات وبرامج الزحف (Google, Bing) لئلا ينكشف
  • يستخدم 8 RPC endpoints مختلفة لضمان الاتصال بالبلوكتشين
  • المحتوى الضار يُجلب ديناميكيًا من العقد الذكي — مما يصعّب اكتشافه

الإصلاح:
  • تم تعطيل إضافة Speed Optimizer
  • تم تعطيل إضافة WP Matrix Authority (إضافة الإخفاء)
  • تم تطهير قاعدة البيانات من الإضافات الضارة
  • تم مسح OPcache على الخادم
  • تم التأكد من خلو الموقع من أي أثر للمالوير بنسبة 100%
